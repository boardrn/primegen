# Prime-Gen Solutions Website

Static one-page website built for GitHub Pages.

## Files

- `index.html` — full one-page website
- `style.css` — responsive styling
- `script.js` — mobile navigation and header behavior
- `assets/logo.svg` — editable SVG logo
- `assets/favicon.svg` — browser icon
- `CNAME` — custom domain placeholder set to `primegen.ca`

## Before publishing

1. Replace the Google Reviews button `href="#"` in `index.html` with the real Google Business review URL.
2. Replace the Facebook button `href="#"` in `index.html` with the real Facebook page URL.
3. If you purchase a different domain, edit `CNAME` to match the exact domain.
4. If you get real jobsite or truck photos later, add them to `/assets` and swap the CSS placeholder sections for real images.

## GitHub Pages setup

Create a GitHub repository, upload all files, then go to:

`Settings → Pages → Deploy from a branch → main → /root`

Add your custom domain in:

`Settings → Pages → Custom domain`

Use the same domain as the `CNAME` file.
